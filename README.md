## The Smart Billing System

A simple, command-line billing application designed for a fruit shop. This project focuses on fundamental programming concepts including string manipulation, type casting, arithmetic operators, and conditional logic.

--------

## Project Overview

This software interacts with the user to collect order details, calculates totals with conditional discounts, and generates a professionally formatted receipt. It is designed to be user-friendly and robust against common input errors.

-------------

## Features

- User Personalization: Automatically formats user names (removes extra spaces, capitalizes the first letter).
- Dynamic Calculations: Calculates total cost based on quantity and price per kg.
- Smart Discount Logic: Automatically applies a 10% discount if the total bill exceeds $500.
- Input Validation: Prevents processing if the quantity entered is zero or negative.
- Professional Receipt: Generates a clean invoice with the item name in ALL CAPS and currency formatted to 2 decimal places.
